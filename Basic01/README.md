# Currency Converter

A simple and intuitive currency converter built with React and Tailwind CSS. This app allows users to convert between USD, EUR, and GBP with real-time calculations. 


Features-

1) Convert Currencies: Convert between USD, EUR, and GBP.

2) Swap Currencies: Easily swap the "From" and "To" currencies with a single click.

3) Dynamic Calculation: Automatically calculates the converted amount based on the selected currencies and input value.


Technologies Used-

1) React: A JavaScript library for building user interfaces.

2) Tailwind CSS: A utility-first CSS framework for styling.

3) JavaScript: For logic and functionality.

4) HTML: For structuring the app.
